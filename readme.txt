== Installation ==

1. Download the Granular zip file and unzip it.
2. Upload the Granular folder to your `/wp-content/plugins/` directory. Alternatively, use the Wordpress plugin install in `Plugins >> Add New >> Upload` to upload and install the zip file.
3. Activate the plugin through the `Plugins` menu in WordPress.
4. Create a `composer.json` file on your theme or plugin and good development.

== Frequently Asked Questions ==

= Can I use other package ? =
 
Yes, simply use composer for your dependencies.

= Can I embed Granular into my plugin ? =
 
Yes, simply copy all Granular files into your project.
